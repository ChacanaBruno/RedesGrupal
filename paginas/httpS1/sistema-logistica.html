Acceso al Sistema de Gestión Logística

Bienvenido al Sistema de Gestión Logística de FAMA SRL. Este sistema permite realizar seguimiento de envíos, gestionar flota, programar entregas y administrar inventarios en tiempo real.

Iniciar Sesión




Si no tiene credenciales de acceso o ha olvidado su contraseña, por favor contacte al administrador del sistema al interno 2345.

Requisitos del Sistema

  • Navegador web actualizado (Chrome, Firefox, Edge o Safari)
  • Conexión a Internet estable
  • Resolución de pantalla mínima: 1024x768
  • JavaScript habilitado

Módulos del Sistema

  • Seguimiento de Envíos
  • Gestión de Flota
  • Programación de Entregas
  • Administración de Inventarios
  • Reportes y Estadísticas
  • Gestión de Documentación
  • Comunicación con Clientes
